# Styleguide: Lestari Kopi

**Konsep:** Alam, Tenang, Tradisional Modern.

## 1. Color Palette

- **--bg-main:** `#F4F1EA` (Cream / Bone) - Warna dasar yang hangat.
- **--bg-surface:** `#FFFFFF` (White) - Untuk kartu dan elemen kontras.
- **--text-primary:** `#3E3B36` (Charcoal) - Teks utama, lebih lembut dari hitam murni.
- **--text-secondary:** `#8C8881` (Stone Grey) - Teks pendukung.
- **--accent-brown:** `#A47551` (Warm Clay) - Warna tombol dan highlight utama.
- **--accent-green:** `#5C7C57` (Sage Green) - Aksen dekoratif alami.

## 2. Typography

- **Headings:** 'Merriweather', serif. Memberikan kesan klasik dan terpercaya.
- **Body:** 'Open Sans', sans-serif. Modern dan mudah dibaca.

## 3. Layout Grid

- **Mobile (<768px):** 1 Kolom. Margin 16px.
- **Tablet (768px - 1024px):** 2 Kolom. Margin 24px.
- **Desktop (>1024px):** 3 Kolom. Max-width 1200px.

## 4. Components

### Buttons
- Rounded corner 4px.
- Flat design.
- Hover effect: darken background.

### Cards
- Border radius 8px.
- Subtle shadow: `0 2px 8px rgba(0,0,0,0.05)`.

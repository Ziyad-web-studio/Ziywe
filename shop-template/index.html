Cara Menggunakan Template

1. Download File

Setelah pembelian berhasil, download file ZIP yang diberikan dan ekstrak ke komputer Anda.

2. Buka di VS Code

Klik kanan folder hasil ekstrak, pilih "Open with Code". Pastikan Anda sudah menginstall ekstensi "Live Server".

3. Edit Konten

Buka file index.html. Ubah teks di antara tag HTML. Ganti gambar di folder assets atau ubah src gambar.

4. Preview

Klik kanan pada index.html dan pilih "Open with Live Server" untuk melihat hasil perubahan secara real-time.

5. Deploy

Upload kode Anda ke GitHub dan aktifkan GitHub Pages, atau upload ke hosting cPanel/Vercel/Netlify Anda.
